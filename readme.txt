Add scroll-based Spline 3D animations inside Elementor using timeline controls.

== Description ==
Wp Spline Animator for Elementor lets you embed Spline scenes and animate them using scroll-based timelines directly in Elementor.

Features:
* Scroll animation timeline
* Scale, position and rotation control
* Elementor widget integration
* Default scroll animation fallback
* Lightweight and responsive

== Installation ==
1. Upload plugin folder to /wp-content/plugins/
2. Activate plugin
3. Use Elementor widget
